# NEMESIS

Deploy autonomous agents on Base. Built on Hermes, executed through Base MCP. Approval-first, always.

![NEMESIS](/assets/nemesis-banner-dark.png)

## Overview

NEMESIS is a Quant-Grade infrastructure platform for deploying autonomous trading agents. 
Our agents monitor on-chain metrics, execute specific strategies (e.g., Auto-Compound, Dip Snipe, New Listings), and wait for your explicit approval via Telegram before any transaction is executed.

**Core Philosophy: Approval-First, Always.**
Your funds are never held in custody. Transactions are generated by the agent, but they will never execute until you tap "Approve" on your linked Telegram device.

## Features

- **Quant-Grade Infrastructure**: Monolith deployment via Railway powered by a high-performance Supabase PostgreSQL database.
- **Approval-First Execution**: Agents simulate and build transactions via Base MCP, but require real-time Telegram approval to broadcast.
- **Telegram Linking**: Secure one-to-one mapping between your Ethereum Wallet and your Telegram Chat ID.
- **Real-Time Monitoring**: Live dashboard for tracking agent status, proposals, and execution history.

## Getting Started

### 1. Database Setup (Supabase)
1. Create a new Supabase project.
2. Navigate to Database settings and copy your `DATABASE_URL` (Transaction pooler URL is recommended).
3. The application will automatically create the required schema (`users`, `agents`, `proposals`, `link_codes`) on first boot.

### 2. Deployment (Railway)
NEMESIS is designed to be deployed as a monolith service using Railway.

1. Connect your GitHub repository to Railway.
2. In the Railway Variables tab, add the following environment variables:
   - `DATABASE_URL` - Your Supabase connection string.
   - `TELEGRAM_BOT_TOKEN` - Your bot token from @BotFather.
   - `NEXT_PUBLIC_SITE_URL` - Your public Railway domain.
3. Deploy! Railway will concurrently run both the Next.js Web App and the Telegram Bot.

### 3. Telegram Linking Flow
Once your service is live:
1. Open the web dashboard and connect your wallet.
2. Click **Generate code** in the Connect Telegram card.
3. Send `/link CODE` to your bot on Telegram.
4. Your bot will confirm the link. All future proposals generated by your agents will now be sent directly to this chat.

## Commands

Control your agents directly from Telegram:
- `/start` - Start the bot
- `/link <code>` - Link your wallet to your Telegram
- `/unlink` - Unlink your wallet
- `/agents` - List all your active and paused agents
- `/status` - Check global system status
- `/pause <id>` - Pause an active agent
- `/resume <id>` - Resume a paused agent
